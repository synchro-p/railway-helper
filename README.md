## Начало разарботки

Скачать [node version manager](https://github.com/nvm-sh/nvm)

```bash
# установить нужную версию node
$ nvm use

# установить зависимости
$ npm ci

# запустить сайт в режиме разработки
$ npm run dev

# собрать продакшн версию статики
$ npm run build
```
